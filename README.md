## Features

- Will open within VSCode - don't lose context!
- Documentation for 20+ libraries (Typescript, React, Express, Tailwind, Jest ++)
- Open with:
  - Mac: `cmd + shift + i`
  - Windows: `ctrl + shift + i`
- Keyboard navigation
